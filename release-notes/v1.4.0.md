# Life Plan Lite v1.4.0

Life Plan Lite `v1.4.0` is the next stable minor release.

This release focuses on making the app more capable for real day-to-day use while also making it easier to learn and easier to grow safely.

## Highlights

- Added a real Help surface with guided tour launch, searchable help topics, and browsable reference articles.
- Added board-view column sorting so lists can be sorted directly from the live board without changing their saved structural sort.
- Expanded Weather widgets with `Current location` and `Custom location` modes.
- Added Wishlist recommendation logic with `Priority`, `Pieces`, `Total Cost`, `Buy Score`, and `Advised Buy Order`.
- Added the first high-level `Project` list template with hierarchy, milestones, timeline boundaries, and a lightweight Gantt view.

## Included Improvements

- Help now includes topic pages for core workspace areas, templates, widgets, summaries, deadlines, and Wishlist logic.
- Wishlist scoring now uses built-in recommendation profiles and handles missing values neutrally instead of failing closed.
- Existing Wishlist lists are upgraded forward automatically with the new fields.
- Project lists support nested task structure, milestone-style checkpoint items, dependency-aware ordering, and timeline consistency prompts.
- Project boundary items (`Project Start` / `Project End`) control the overall project timeline without cluttering the visible board rows.
- Milestones now display their dependency context directly and auto-align their planned date with the latest dependency completion when needed.
- Clock, Countdown, and World Clock widgets were unified into a cleaner shared digital design family.
- Weather reliability and custom-location support now work in both development and packaged builds.
- Help and tutorial content were extracted out of the main renderer file enough to remove Babel's large-file deoptimization warning on `App.tsx`.

## Release Focus

`v1.4.0` keeps the product firmly local-first, but it raises the floor meaningfully:

- the app is easier to understand
- the board is easier to interact with directly
- Wishlist now supports practical purchase planning
- Project lists now provide a real high-level project overview without turning LPL into a full PM tool

This release also sets up a healthier foundation for the `v2.x` feature line by tightening documentation, planning, and renderer structure.

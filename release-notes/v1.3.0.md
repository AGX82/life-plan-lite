# Life Plan Lite v1.3.0

Life Plan Lite `v1.3.0` is now the latest stable release.

This release focuses on polish, onboarding clarity, and making the first-use experience feel complete and confident.

## Highlights

- Refined the guided tutorial so it teaches the real app workflow more clearly
- Added a fuller walkthrough that covers navigation, edit-panel scope, list tabs, layout, settings, and the live board
- Improved tutorial pacing, spotlight behavior, and transition sequencing for a calmer feel
- Added an explicit handoff from admin/editor mode into the live board view
- Replaced sparse tutorial content with a richer synthetic walkthrough board that feels like a real planning setup
- Improved tutorial targeting so list and widget sections are highlighted as full sections, not just headers
- Kept the tutorial reusable at any time from the left rail

## Included Improvements

- Softer backdrop treatment during tutorial steps so the surrounding interface remains legible enough to build context
- Better spotlight framing for list-tab walkthrough steps
- Stable tutorial experience even when the user's actual boards are empty or minimal
- Cleaner left-rail action grouping around board display controls
- Preserved reset behavior where the app returns to one empty base board instead of zero boards

## Release Focus

`v1.3.0` is meant to feel more welcoming and more teachable without changing the core local-first planning model. The editor remains the place for setup and structural change, while the live board remains the primary day-to-day surface.

# Life Plan Lite v1.0.1 Hotfix Notes

Release date: 2026-04-28

This is a patch release for `v1.0.0` focused on stabilizing the first-run configuration wizard.

## Fixed

- Fixed a wizard failure path where adding multiple widgets during first setup could raise a layout error even when the board should have had enough room.
- Fixed wizard layout validation so newly created lists and widgets are checked together before being shown on the board, preventing hidden overlap bugs.
- Fixed a case where a list could end up placed on top of a widget after wizard completion.
- Fixed a failed new-board wizard run leaving behind a partial inactive board.
- Fixed a failed first-run/reset wizard run leaving the reused starter board in a half-applied state.
- Fixed page #3 so adding another list of the same type inserts it directly under that type instead of appending it to the bottom of the full list.
- Fixed page #4 radio controls so they render as normal radio buttons instead of oversized boxed inputs.
- Fixed wizard-created World Clock defaults so their initial size matches the layout assumptions used during wizard placement.

## Scope

- No data model reset is required for existing users.
- This release is intended as a safe replacement for `v1.0.0`.

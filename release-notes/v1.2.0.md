# Life Plan Lite v1.2.0

Life Plan Lite `v1.2.0` is the first minor release after the initial `v1.0.x` stabilization cycle. This release turns the onboarding and fast-setup experience into a first-class part of the app.

## Highlights

- Added the full `LPL Wizard` flow for first run, quick-add list creation, new-board creation, and reset-to-first-run.
- Added the in-app guided tutorial, with reopen support from the left rail and walkthrough coverage for the main admin workflow.
- Improved wizard layout planning so list and widget placement is more intentional and resilient.
- Improved reset behavior and close handling so the app can be intentionally left in a clean empty state with one base board ready for manual setup.

## Wizard Improvements

- Users can create several lists in one pass instead of configuring each list tab-by-tab.
- Page 3 now inserts additional lists directly under the matching list type.
- Shopping-specific and birthday-specific wizard options are supported as part of setup.
- Widget rows support add, remove, layout choice, and show/hide before finishing.
- If the wizard cannot fit every selected visible element on the board, it now:
  - creates the lists/widgets anyway
  - hides the ones that do not fit
  - explains which ones were created but are not currently displayed

## Tutorial And Onboarding

- Added a guided tutorial overlay that explains:
  - boards list
  - wizard access
  - board tree
  - edit panel scope
  - list edit tabs
  - live layout
  - app settings
  - display targeting
  - fullscreen board display
- Tutorial pacing and spotlight movement were refined for a calmer, easier-to-follow rhythm.
- List-tab tutorial steps now highlight the full active tab content, not only the tab label.

## Reset And First-Run Behavior

- Reset now requires confirmation before clearing data.
- After reset is confirmed, the wizard can be closed directly from the first rebuilt screen.
- Reset leaves one empty `Life Plan Lite` board as the intentional empty-state baseline.

## Layout And UX Refinements

- Wizard board planning prioritizes:
  - To Do lists on the top-left
  - widgets in the top-right strip
  - fixed-size template lists before flexible ones
  - flexible list widths based on visible field density
- Added Back navigation on wizard screens after the first.
- Polished final-screen wording and wizard footer actions.

## Stability Notes

- Built and packaged as the portable Windows executable.
- Verified through typecheck, production build, and Windows packaging.

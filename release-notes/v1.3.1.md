# Life Plan Lite v1.3.1

This is a hotfix release for the `v1.3.x` line.

## Fixed

- Weather widget now works reliably in packaged Windows builds by moving fallback location lookup and weather retrieval onto the desktop-side Electron bridge.
- Board top summaries now size more naturally to their actual content instead of wasting space in overly wide slots.
- Summary labels are easier to read, and summary values now use semantic coloring:
  - positive indicators such as counts stay green
  - alert indicators such as purchases and overdue counts render in red
- Total Purchases board summaries now display rounded whole-number totals by currency, without decimals.

## Notes

- This release is a targeted polish/stability update on top of `v1.3.0`.
- Ongoing feature development continues from the `1.3.2-dev` line after this tag.
